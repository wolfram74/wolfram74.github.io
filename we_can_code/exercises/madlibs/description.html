Mad Libs

Description

One joke is that the internet is not a series of tubes, but a series of forms. Log-in forms, comment forms, shopping order forms, just forms everywhere. This exercise is a small introduction to doing arbitrary operations with form data using javascript and the jquery library.

Resources

Go to this repository, fork it, download the files and set it up as a page on your github.io repo.

Examples

Here's a working example of the mad libs game, play it a few times, explore the HTML but not the javascript and try to implement the functionality.
