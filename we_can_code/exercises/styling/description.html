Page Layout

Description

A web page is constructed of HTML (hypertext markup language) which informs the browser how information and content is related to each other. Beyond a basic level, it's appearance is governed by CSS (cascading style sheets) which can produce a bewildering array of visual effects. This exercise provides you with a design object (the Berkshire Hathaway webpage) and 2 different starting points for HTML. Both have been stripped of their meaning. Go over the readme in the project for more details.

Resources

Go to this repository, fork it, download the files and set it up as a page on your github.io repo.
