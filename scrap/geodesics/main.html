problem statement: 6.07, given a cylinder of radius R, find the shortest path from $&\theta_1, z_1&$ to $&\theta_2, z_2&$

  1. $&\frac{\delta f}{\delta y}-\frac{d}{dx}\frac{\delta f}{\delta y'}=0&$

    Euler lagrange equation, if x is an independent variable that defines y, and f is a curve defined by x, y and $&y'=\frac{dy}{dx}&$ then satisfying that differential equation means f is a stationary solution to $&\int_{p_1}^{p_2}fdx&$, some kind of local extrema or saddle point.

  2. $&ds = \sqrt{\Sigma_1^k (de_i)^2}&$

    The infinitesimal displacement in a space with k orthonormal basis vectors is calculable as a pythagorean-ish displacement of the infinitesimal displacements in the k dimensions (is this referred to as a euclidean metric?)

  3. $&ds = \sqrt{dz^2 + R^2d\theta^2}&$

    z being a normal euclidean coordinate has no special form for it's infinitesimal form, $&\theta&$ being an angle measurement needs the distance from the axis of rotation R to figure define it's infinitesimal displacement

  4. $&\int_{p_1}^{p_2}ds=\int_{p_1}^{p_2}\sqrt{dz^2 + R^2d\theta^2}&$

    An expression for length between two arbitrary points, at this point it doesn't seem a non-straight line solution is permissible.

  5. $&\frac{d\theta}{dz}=\theta' \implies d\theta = \theta' dz&$

    because it's a full derivative as opposed to a partial derivative ($&\frac{\delta \theta}{\delta z}&$) the infinitesimal portions can be separated and manipulated as though they are free standing variables. This irritates some mathematicians?

  6. $&\int_{p_1}^{p_2}\sqrt{dz^2 + R^2d\theta^2}= \int_{z_1}^{z_2}\sqrt{dz^2 + R^2dz^2\theta'^2}= \int_{z_1}^{z_2}\sqrt{1 + R^2\theta'^2}dz= R\int_{z_1}^{z_2}\sqrt{1/R^2 + \theta'^2}dz&$

    Using the result from 5 to turn a multivariable curve into a curve of the form described in 1

  7. $&f(\theta, \theta', z) = \sqrt{1 + R^2\theta'^2} \implies \frac{\delta f}{\delta \theta}-\frac{d}{dz}\frac{\delta f}{\delta \theta'}= 0 - \frac{d}{dz} \frac{R\theta'}{\sqrt{1/R^2 + \theta'^2}}=0 \implies \frac{R\theta'}{\sqrt{1/R^2 + \theta'^2}} = C&$

    The euler lagrange equation dictates that curves that will generate stationary values for length of paths between two points on a cylinder, when differentiated with respect to $&\theta'&$, will have no z dependence

  8. $&\frac{R\theta'}{\sqrt{1/R^2 + \theta'^2}} = C_1 \implies \frac{R^2\theta'^2}{1/R^2 + \theta'^2} = C_2 \implies R^2\theta'^2 = C_2/R^2 + C_2 \theta'^2 \implies \\ (R^2-C_2)\theta'^2 = C_2/R^2 \implies \theta'^2 = \frac{C_2/R^2}{R^2-C_2} \implies \theta' = \sqrt{ \frac{C_2/R^2}{R^2-C_2} } = C_3 \implies \theta(z) = m z + C_3&$

    Which is concerning because it implies at $&R^2 < C_2&$ the slope is complex. Otherwise it seems like a valid solution.
