There are several built in ways of storing and representing data in Ruby, Hashes, Arrays, Strings and Integers being among them. We'll describe some basic methods using them, the convention we'll use is the object in an example will be lower cased.

integer = 0
Creating an integer by assigning an integer value to a variable.
integer += other_integer
Incrementing an integer by some other integer amount, the same formatting can be applied to other arithmetical operations like -, *, /, ** (exponentiation) and % (modulus)
integer % other_integer
Determines the remainder of the division operator.
string = String.new()
Explicitly creating a new string, will be empty
string = "an example"
Implicitly creating a new string, will start as the input string.
string += another_string
Concatenates another string onto the first string.
string[index] = new_string
replaces the character at index, with the new string, effects only the character at the index.
array = Array.new()
Explicitly creates a new array, will be empty by default.
array = []
Implicitly creates a new empty array.
array << item
Inserts an item onto the back of the array.
array += other_array
concatenates another array onto the back of the first array.
array[index]
Returns the value of the array at that index.
array[index]= new_value
resets the value of the array at that index.
hash = Hash.new()
Explicitly creates a new hash, empty by default.
hash = {}
Implicitly creates an empty hash.
hash = {key => value}
creates a key-value pair in a hash.
hash[key]
accessing the value of a hash assigned to a particular key. (runs in constant time, very useful.)
hash[key] = new_value
Assigns a new value to a hash with that key.
