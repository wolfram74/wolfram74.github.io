We Can Code Workshop, Iowa City

Locally sourced exercises

  • Mad Libs

     A mimicry exercise to practice DOM discovery, event listeners and working with form elements

  • Page Layout

     A mimicry exercise to practice DOM construction, CSS and HTML structure.

  • Basic controls

     A mimicry exercise to practice canvas manipulation, key input and simple state management.

  • From 0 to basic games

     An article loosely outlining the base components of a game and what kinds of javascript knowledge allows you to make them.

External Resources

Github

Github is a service that will offer remote version control based on the open source software, git. Has a complimentary tier, and is a major part of contributing to open source projects and participating in many software communities

Learn Enough Git to Be Dangerous

A very thorough book on the subject of git version control.

HackerRank

HackerRank is a web site that provides a large collection of algorithm and computer science related challenges. It can be a good place to practice and compare coding techniques. Supports log in with github. Challenges can be completed in a wide variety of languages

Nature of Code

A detailed walk through of using software to simulate things found in nature. While code examples are given in the language Processing, they can be done in Javascript after familiarizing yourself with how the canvas html element works.

What most young programmers need to learn

An opinion piece on some bad habits that are easy to develop when first starting to learn programming.

Python Exercises for Kids

A collection of relatively self contained projects that, while recommended for python, can be implemented in any language roughly equally well. Turtle would have to be substituted for canvas in Javascript, for example.

Javascript Koans

Koans are test driven muscle memory exercises that introduce the syntax of a language in a hands on approach. First done with Ruby, many people have made koans for other languages including Javascript. mrdavidlain and liammclennan both have implemented different sets of koans for Javascript.

Diving Into Other People's Code

An in depth first hand recounting of contributing to a large preexisting open source code base.

Codecademy

A collection of muscle memory type exercises in various languages and techniques.

Call me Nick tutorials

A large collection of tutorials and example projects in front end design.

Three Basic Pillars to Develop Simple and Appealing Games

A broad strokes over view of one approach to game design.

I know how to program, but I don't know what to program

An article going over the thought process one might have when trying to come up with intentional practice for programming.

Don't Think Big!

An explanation on how projects with too large of scope can consume effort in research and set up that could have been used in creativity. Also applies to practice projects.

Code Kata

A collection of larger scale exercises intended to be discussed in groups and deliberated over.

Javascript 30

A collection of small projects that cover a broad base of javascript skills.
